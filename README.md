# Pathfinder Viz

> Lab project — visual exploration of pathfinding algorithms and maze generation.

Interactive maze and pathfinding algorithm visualizer built with Next.js, TypeScript, and Tailwind CSS.

## Algorithms

- **BFS** - Breadth-First Search (guarantees shortest path)
- **DFS** - Depth-First Search (explores deep before wide)
- **Dijkstra** - Dijkstra's algorithm (weighted shortest path)
- **A*** - A* with Manhattan heuristic (informed search)

## Features

- Clickable grid to draw walls
- Drag to paint multiple walls
- Recursive backtracking maze generation
- Adjustable speed and grid size
- Color-coded visualization (walls, visited, frontier, path)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
